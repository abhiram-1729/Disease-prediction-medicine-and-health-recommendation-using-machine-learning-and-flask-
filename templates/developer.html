About Us

Hello, we are Rangoon Abhiram, Duddula Rachana, and Salkuti Divya Reddy—passionate developers and data enthusiasts. Together, we’ve built a project titled 'Disease Prediction and Medicine Recommendation System using Machine Learning and Flask.' As a team, we bring a strong blend of skills in Python, Flask, Machine Learning, Data Analysis, and Frontend Development. With over years of combined experience in the tech industry, we are driven by a shared vision: to harness the power of technology to improve healthcare accessibility and support early diagnosis. This project showcases our ability to build full-stack web applications powered by intelligent algorithms. Users can input symptoms through a clean and interactive frontend, which we developed with a focus on usability. The backend, built using Flask, processes the data and communicates with a machine learning model trained on medical datasets to accurately predict possible diseases. Based on the prediction, the system recommends relevant medicines and general wellness suggestions. Our goal was to create a system that not only predicts but also empowers users with actionable insights—demonstrating how data-driven solutions can enhance healthcare decision-making.
